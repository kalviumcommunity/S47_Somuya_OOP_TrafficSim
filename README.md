# TrafficSim

## Project Description

**TrafficSim** is a simple traffic management simulation developed for the Object-Oriented Programming course. It demonstrates the basic concepts of classes and objects by simulating vehicles and traffic lights in an urban traffic environment.

## Features

- Simulates vehicle movement and traffic light changes
- Demonstrates basic OOP concepts such as classes, objects, and encapsulation
- Adjusts vehicle speed and traffic light states
